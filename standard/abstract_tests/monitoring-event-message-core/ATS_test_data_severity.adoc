[%metadata]
label:: /conf/event-message-core/data_severity
subject:: /req/event-message-core/data_severity
test-purpose:: Validate that a WMEM has a valid data severity.

[.component,class=test method]
=====
[.component,class=step]
--
Check for the existence of a `+data.severity+` property in the WMEM.
--

[.component,class=step]
--
Check that the `+data.wmem-severity+` property is equal to one of the following values:

* ``DEBUG``
* ``INFO``
* ``WARNING``
* ``ERROR``
* ``CRITICAL``
--
