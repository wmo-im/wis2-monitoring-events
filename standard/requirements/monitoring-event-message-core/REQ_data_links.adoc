[[req_monitoring-event-message-core_data_links]]
[width="90%",cols="2,6a"]
|===
^|*Requirement {counter:req-id}* |*/req/monitoring-event-message-core/data_links*
^|A |A WMEM SHALL provide a `+data.links+` array property, when a `+data.content+` property is not provided.
^|B |The `+links+` array property SHALL contain at least one link with, at a minimum, the `+href+` and `+rel+` properties.
^|C |The links SHALL be HTTP, HTTPS, FTP or SFTP.
|===
