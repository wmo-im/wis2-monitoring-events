^|*Requirement {counter:req-id}* |*/req/monitoring-event-message-core/data_severity*
^|A |A WMEM SHALL provide a `+data.severity+` property.
^|B |The `+data.severity+` property SHALL be one of the following values:

* ``DEBUG``
* ``INFO``
* ``WARNING``
* ``ERROR``
* ``CRITICAL``
